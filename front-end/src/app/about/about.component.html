What's this?

This application allows you to write Julia code and attempt to convert it into something useable on the web. Currently it does this through one of two approaches:

  • Converting to Javascript via Tom Short's ExportWebAssembly.jl and Emscripten.
  • Converting to WebAssembly via Mike Innes's Charlotte.jl (with contributions from Tom). The resulting WebAssembly text format is converted to a corresponding binary file format via the WebAssembly Binary Toolkit.

After each tool converts julia to either javascript or wasm it loads it into the browser window so that you can test it out. The API for the Charlotte.jl tool is written in HTTP.jl with some help from Joseki.jl.
